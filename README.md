# Project Overview

This project demonstrates the development and deployment of a medical imaging AI system, focusing on performance efficiency, scalability, and real-world usability. It includes a REST API for batch inference and a Streamlit-based frontend for interactive multi-image prediction and reporting.

# Features

- CNN-Based Image Classification: Pneumonia detection from chest X-ray images
- CPU-Optimized Inference: Designed to run efficiently without GPUs
- Batch Image Processing: Upload and analyze multiple X-rays at once
- REST API: Flask-based API for scalable inference
- Interactive UI: Streamlit frontend for medical image analysis
- Export Options: CSV and JSON report generation

# Technologies Used

- Programming Language: Python
- Deep Learning: TensorFlow / Keras
- Image Processing: OpenCV
- Model Architecture: Convolutional Neural Networks (CNN)
- Backend/API: Flask
- Frontend/UI: Streamlit

# Output

- Pneumonia prediction results for uploaded X-ray images
- Structured medical reports
- Exportable prediction data in CSV and JSON formats
